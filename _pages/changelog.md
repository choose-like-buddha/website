# Changelog
Stay up to date with the latest improvements, features, and fixes in Choose Like Buddha, a mindfulness and decision-making app inspired by Buddhist philosophy. On this page, we share detailed release notes for each version, including new features, performance enhancements, and bug fixes.

We regularly update the app to improve user experience, add meaningful tools like daily reminders, and ensure stability across iOS devices. Check back often to see how the app evolves and how each update helps you make calmer, more conscious choices in everyday life.
<br>

### `Latest`
## Version 1.5 - January 29, 2026
### Subscription Improvements & Stability Update

- Updated **Premium subscription implementation** to improve reliability and resolve purchase errors  
- Improved handling of subscription status across app sessions  
- Enhanced purchase flow clarity for Free Path and Enlightened Path users  
- Fixed edge-case bugs related to premium access  
- General performance and stability improvements  

This release ensures a smoother and more reliable subscription experience for all users.

---

## Version 1.4 — January 25, 2026  
### Daily Reminders & Practice Consistency

- Introduced **daily reminder notifications** to encourage regular mindfulness practice  
- Improved reliability of daily question delivery  
- Fixed subscription-related edge cases  
- General performance and stability improvements  

This update helps users stay consistent with daily reflection and mindful decision-making.

---

## Version 1.3  
### Breathing Tool & Emotional Awareness

- Added **4-4-4 guided breathing tool** for calming the mind and reducing stress  
- Introduced **food–body feeling tracking**, allowing users to note how specific foods affect their body and mood  
- Improved navigation between tools and daily questions  
- Minor UI refinements  

Expanded the app beyond questions into practical mindfulness tools.

---

## Version 1.2  
### Progress Tracking & Personalization

- Added **progress chart** showing correct answers per day  
- Introduced **user accounts** with email and Apple Sign-In  
- Enabled users to **set and edit their name**  
- Improved data persistence across sessions  

This version made progress measurable and the experience more personal.

---

## Version 1.1  
### Premium Access & Usage Limits

- Introduced **Premium subscription** (monthly)  
- Free Path: up to **5 questions per day**  
- Enlightened Path (Premium): increased daily question limit  
- Improved explanation clarity after each answer  

Established sustainable access to extended daily practice.

---

## Version 1.0  
### Initial Release

- Launched **Choose Like Buddha** on the App Store  
- Daily quiz-style questions based on mindful and ethical decision-making  
- Clear explanations provided after each answer  
- Calm, minimalist design focused on reflection  

The foundation of a mindful decision-making app rooted in awareness and wisdom.
